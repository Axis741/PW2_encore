## Objetivo del Proyecto

Desarrollar una plataforma web donde:

- Los usuarios puedan explorar productos.
- Visualizar información de la mercancía.
- Simular una experiencia de compra.
- Un administrador pueda gestionar productos (agregar, editar o eliminar).

---

## Ramas del Repositorio

El proyecto se gestiona mediante el uso de ramas en GitHub para mantener un mejor control del desarrollo y organización del código.

Actualmente cuenta con las siguientes ramas:

- `main`  
  Es la rama principal del repositorio. Aquí se integran las versiones finales y completas de nuestro proyecto.  
  Funciona como la versión base o de entrega final.

- `master`  
  Contiene el desarrollo actual de las pantallas del sistema.  
  En esta rama se trabajan los archivos `.html`, los estilos `.css` y los recursos visuales.  
  Se utiliza como rama de desarrollo antes de integrar los cambios finales a `main`.

El uso de estas ramas nos permite trabajar de manera más organizada, evitar conflictos en el código mientras seguimos desarrollando nuevas funcionalidades.
